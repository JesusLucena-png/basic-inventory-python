# Simple Product Inventory Program

## Description
This project is a simple Python inventory program.  
It is a console-based application that stores product information entered by the user.

The program is designed to demonstrate basic Python concepts such as lists, dictionaries, loops, and error handling.

## How it works
1. The program asks the user to enter a product name.  
2. It requests the unit price and the product quantity.  
3. The system calculates the total cost using price multiplied by quantity.  
4. The program stores the product data in a list using a dictionary structure.  
5. It prints the product information including name, price, quantity, and total cost.

## Status
> The project is currently running as a basic inventory input system, and it is demonstrating how Python processes user data.
