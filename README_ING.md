# Simple Product Inventory Program

## Name

**Simple Product Inventory Program**

## Description

This project is a simple Python inventory program.
It is a console application where the user can enter product information.

The program stores basic data such as the **product name, price, and quantity**.
It also calculates the **total cost** of the product.

This project demonstrates basic Python concepts like **lists, dictionaries, loops, and error handling**.

## How it Works

First, the program asks the user to **enter the product name**.

Then, it asks for the **unit price** and the **product quantity**.

After that, the program **calculates the total cost** by multiplying the price and the quantity.

Next, the program **stores the product information in a list** using a dictionary.

Finally, the program **prints the product information and the total cost** in the console.

## Usage

1. Run the Python file in the terminal.

2. The program will ask you to enter:

   * Product name
   * Unit price
   * Product quantity

3. Enter the requested information.

4. The program will calculate the **total cost**.

5. Finally, the program will display the **product information and total cost**.

## Outcome

The result is a simple program that can **store product information and calculate the total cost**.

This project helps practice **basic Python programming and user input handling**.
